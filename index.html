03. Projects

Scran Master

Scan Master is a fast React-based takeaway application. It uses Bootstrap 5 for a mobile and desktop responsive and modern look. Scan Master fetches restaurant data from an API. Users can add and remove items from the order. The project can simulate placing an order which will send the data back to the API and then we receive an estimated delivery time for that order.

React | Bootstrap | Postman | API

CarPe Diem

CarPe-Diem is an object-oriented(OOP) PHP application that was built using SOLID principles. The look of the website is built using SASS. The application fetches, hydrates and displayed data about cars from a MySQL database. You can filter the make and colour of the car or just search it by the name. This project was utilising namespacing and autoloading as well as unit testing functions.

PHP | SASS | SQL | HTML

WordRunner

WordRunner is a fun word-guessing game. It fetches a collection of words and hints from an API. The player is presented with a list of synonyms which will be their hints of the right word. The player has a limited amount of time to guess the correct word else its a game over. The goal is to gain a high score.

React | Bootstrap | Postman | API
